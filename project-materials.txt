*Proyek:* Membuat Sistem Pembayaran Online dengan API dan Backend

*Deskripsi:* Dalam proyek ini, kita akan membuat sebuah sistem pembayaran online yang menggunakan API dan backend untuk mengelola proses pembayaran.

*Video Tutorial:* Lihat video tutorial lengkapnya di YouTube: Segera Hadir

*Bahan-bahan yang digunakan:*

*Konfigurasi*
- `config.php`: File konfigurasi utama yang berisi pengaturan API dan produk.

*Sumber Daya*
- QR Code Contoh: https://forestapi.web.id/qr/b7c88Gn9otHRlj61thT3
- Thumbnail Produk Contoh: https://forestapi.web.id/home/images/project-wa-bot-topup.jpg

*Framework dan Library*
- Tailwind CSS: https://cdn.tailwindcss.com (digunakan untuk styling halaman)
- Font Awesome: https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.2/css/all.min.css (digunakan untuk ikon)
- Google Fonts: https://fonts.googleapis.com/css2?family=Inter:wght@400;600;700&display=swap (digunakan untuk font "Inter")

*API dan Endpoints*
- Dokumentasi API: https://forestapi.web.id/api-documentation
- Endpoints:
- `/deposit/create`: Membuat pembayaran baru
- `/deposit/status`: Mengecek status pembayaran
- `/deposit/cancel`: Membatalkan pembayaran

*File Backend*
- `download.php`: Skrip backend untuk menangani proses pengunduhan file setelah pembayaran berhasil.
